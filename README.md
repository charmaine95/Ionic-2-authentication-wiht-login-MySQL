การทำ Ionic-2-authentication-wiht-login-MySQL

ทดสอบใช้งาน Project
$ git clone https://github.com/jeynever/Ionic-2-authentication-wiht-login-MySQL.git
$ cd ไปที่ .../.../Ionic-2-authentication-wiht-login-MySQL (เลือก PATH ให้ถูกด้วย)

$ ionic serve
